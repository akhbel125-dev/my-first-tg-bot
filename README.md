# 🚀 AI Telegram Bot-Portfolio / Бот-Визитка Разработчика

Интерактивный Telegram-бот со встроенным искусственным интеллектом (Llama 3.1 via Groq API), системой управления (FSM) для автоматического сбора технических заданий от клиентов и полноценной админ-панелью для владельца.

## 🛠 Технологический стек:
* **Язык разработки:** Python 3.11+
* **Фреймворк:** `aiogram` (v3.x) — асинхронная библиотека для работы с Telegram Bot API
* **Интеграция с ИИ:** `groq` (модель `llama-3.1-8b-instant`)
* **Архитектура:** Фильтры, Хэндлеры, Finite State Machine (FSM) для пошаговых опросов
* **База данных:** Файловое хранилище уникальных ID пользователей для рассылок
* **Хостинг/Деплой:** GitHub + Render (Web Service)

## ⚡️ Основной функционал:

### Для пользователей:
1. **Умный ИИ-ассистент:** Бот обрабатывает любые текстовые сообщения пользователя в реальном времени, консультирует по услугам разработки, преимуществам автоматизации и мотивирует сделать заказ.
2. **Интерактивный конструктор заказа (Квиз):** Пошаговая форма заказа через FSM. Бот собирает информацию о сфере бизнеса, необходимых функциях и бюджете клиента без использования сложных терминов.
3. **Портфолио и информация:** Быстрый доступ к информации о разработчике и прямой переход в этот GitHub-репозиторий.

### Для администратора (`/admin`):
* **Защита доступа:** Панель управления доступна строго по `ADMIN_ID`.
* **Живая статистика:** Просмотр общего количества уникальных пользователей, запустивших бота.
* **Массовая рассылка:** Возможность отправить сообщение сразу всем пользователям из базы данных в один клик с защитой от лимитов Telegram (`asyncio.sleep`).
